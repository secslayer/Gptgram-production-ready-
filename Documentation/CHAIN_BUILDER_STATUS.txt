HOW TO USE CHAIN BUILDER:
--------------------------
1. Navigate to: http://localhost:3000/chains

2. Add Nodes:
   - Click "Add Input Node" to add user input
   - Click agents in library to add to canvas
   - Click "Add Moderator" for transformation nodes
   
3. Connect Nodes:
   - Drag from bottom handle (output) to top handle (input)
   - Connections appear as green animated lines
   
4. Edit Input:
   - Click "Edit" on input node
   - Type your text
   - Click "Save"
   
5. Delete Nodes:
   - Click to select node(s)
   - Click "Delete" button or press Delete key
   
6. Save Chain:
   - Click "Save" button
   - Chain saved to browser storage
   
7. Load Chain:
   - Click "Load" button
   - Previously saved chain restored
   
8. Execute Chain:
   - Click "Run Chain" button
   - See execution result panel
   - Check run history at /runs
